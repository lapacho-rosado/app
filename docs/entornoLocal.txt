Descripción General:
--------------------
	
	Las aplicaciones desarrolladas en java se componen principalmente de aplicaciones web (.war), en algunos casos con un backend desacoplado constituido por un módulo EJB y en otros incluyendo su propio backend y de módulos EJB. Eventualmente aplicaciones simples de escritorio principalmente para pruebas.

	El motor de base de datos empleado es PostGres en su versión 9.x, utilizando PgAdmin III como interfase gráfica de administración.

------------------------------------------------------------------------------

Ambiente:
---------
	
	JVM: se utiliza la versión 1.8.0_x
	IDE: NetBeans 7.4 y Eclipse
	Server: Wildfly 10.x
	BD: Postgres 9.x


Tecnología asociada:
--------------------

	Maven: versión 3.3.x para la gestión dedependencias, versiones, arquetipos y proyectos.
	Opcional: CygWin: versión 1.7.28-2 para la habilitación de la terminar del NetBeans para la gestión de Maven.
	Opcional: PgAdmin III para la gestión graáfica del motor de base de datos.
